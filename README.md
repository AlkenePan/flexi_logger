# flexi_logger
A flexible logger that can write to stderr or to log files

## Documentation
See http://emabee.atwebpages.com/rust/flexi_logger/index.html


## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
flexi_logger = "0.1"
```

and this to your crate root:

```rust
extern crate flexi_logger;
```
